# TileOrderManager - Сервис для сбора заказов по производству брусчатки

Сервис автоматизирует сбор заявок от клиентов для производства брусчатки используя Telegram бота.

## Реализовано

Серверная часть решения - DAL, BLL, API слои

## В процессе

Telegram Bot для общения с клиентами

## Запланировано

Веб-сайт - для общения с клиентами используя авторизацию через Telegram, а так же изменение данных в базе администраторами или мастерами
Десктопное приложение - для администраторов и мастеров
Мобильное приложение - для галочки
